                                                                      DEPLOYMENT

Aplikasi web interaktif ini dibina menggunakan Streamlit untuk meramal kebarangkalian seorang pemohon pinjaman mengalami kegagalan bayaran (default). Projek ini menggabungkan analisis data yang mendalam dengan algoritma Machine Learning untuk membantu proses membuat keputusan kewangan yang lebih tepat.

tautan link Aplikasi: [vaninovita.streamlit.app](https://vaninovita.streamlit.app/)
